# Modern SNMP Command Line Utility

This command line utility is a demo application showing how to integrate [SNMP MP](../snmp_mp) and
[SNMP USM](../snmp_usm).

Supported command types:

* Get request
* Get next request
* SNMP walk
* SNMP bulk walk
